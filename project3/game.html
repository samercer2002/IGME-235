Survive the Hoard

Insturctions:

  • Movement: WASD
  • Shooting: Arrow Keys
  • Use Item: Space
Note: The specific arrow key correlates to a direction to fire the bullet

Items:

  • Gun: Increases Fire Rate by 2x of base
  • Coffee: Increases movement speed by 1.5x base (currently red meatball)
  • Cross Hair: Enables piercing shots that go through all enemies in its path (currently cyan meatball)
  • Wheel: Enables 8-Directional shots at once (currently purple meatball)

Goals:

The player should try to survive as long as possible given the 3 lives the player has. There is an initial set of 5 enemies and after that, sets of 3 enemies will spawn every 4.5 seconds.

Occasionally, enemies will drop temporary power-ups that help the player survive longer. If the player has an item already in their inventory, passing over another item will not pick it up. Instead, it will use that item. The list of current items in the game can be seen on the left with their effects.
